OAuth Test Console

Test social login against the Mako API. Serve this page from the API origin or set the base URL below.

API configuration

Default: current page origin. Open at /oauth-test.html when the API serves the public/ folder.

1. Browser redirect flow

Click a provider to start OAuth. After login, the API callback returns JSON (e.g. /api/v1/auth/google/redirect). Copy that JSON and paste it in section 2.

2. Import callback JSON

Paste the JSON response from an OAuth redirect callback to store tokens locally.

3. Token exchange flow

POST a provider access token to the *-auth endpoint (mobile / SPA style).
